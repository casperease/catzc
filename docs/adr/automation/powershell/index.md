# PowerShell automation ADRs

The ADRs here govern how PowerShell automation code is written — the language-specific rules (approved verbs, error handling, formatting,
`.ps1` module files, and the rest). Cross-cutting, language-agnostic platform and module-system rules live one level up, directly under
`automation/`; the full code↔ADR registry is the ADR index at the root of `docs/adr/`.
